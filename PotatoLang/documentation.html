PotatoLang Documentation

Here is a complete documentation for PotatoLang.

Comments.

You always need comments even if the code is self-explainatory! You can start on with $!

Examples:
$ This is a comment.
$ say "Cant read this"
say "You can read this"

Output:
You can read this

Make sure to add a space after the $ or it would error out!!

Print Text

To print text to the screen you can use the keyword say followed by what you want to say.

Example:
say "Where the hay is Waldo?!"

Output:
Where the hay is Waldo?!

You can also print variables by doing the following (floop = "bloop"):
say "Value of 'floop' is:" floop

Output:
Value of 'floop' is: bloop

Create A Variable

To create a variable you use the keyword set followed by the name, type and value of the variable.

Example:
set floop words bloop
set green number 420
set cake decimal 3.14
set isSimple yes/no yes
set bugFree yes/no no
say floop green cake isSimple bugFree

Output:
bloop 420 3.14 yes no

There are a few types of variables you can create:

  • words > Words that can be used later in the program. (Future version will have a string parser.
  • number > Classic numbers, -##### to ####
  • decimal > Decimal point numbers, -####.## to ####.##
  • yes/no > Yes/No, On/Off, True/False... but use yes/no please. :)

Get user input

To get user input you can use the keyword get followed by the name of an existing variable to get and the message for the user.

Example:
set guess number 0
get guess Guess a number between 1 and 10

In the future I am hoping to make it parse the message as a string.

If Statements

To create an if statement you use the keyword if followed by the name of the variable you want to check, the operator you want to use, and the value you want to check against.

Examples:
set num1 number 0
set num2 number 3
set num3 number 3
if num1 equals num2 then say "1 is 2" else say "1 not 2"
if num2 equals num3 then say "2 is 3" ; say "and heres multilines" else say "2 not 3" ; say "and multilines here too."

Output:
1 not 2
2 is 3
and heres multilines

There are a few operators you can use:

  • equals
  • is-not
  • greater-than
  • less-than


Currently it is really basic and only supports equals. possibly bug ridden.

whats

This keyword is used to find information about certain words.

Examples:
set green number 420
set vodka words potato
whatis green vodka

Output:
green is a number with the value of: 420
vodka is a buncha words with the value of: potato

You can use it on any other word that the Interpreter knows and it will tell you about it.
